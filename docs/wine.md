# 와인 기초 가이드

---

# 1. 와인 생산국

## 구대륙 와인 (Old World)

전통적인 와인 생산 역사를 가진 유럽 중심 국가들입니다.
테루아(기후·토양·지역성)와 전통 양조 방식을 중요하게 여깁니다.

| 국가    | 특징                | 대표 지역          |
| ----- | ----------------- | -------------- |
| 프랑스   | 세계 와인 기준이 되는 국가   | 보르도, 부르고뉴, 샹파뉴 |
| 이탈리아  | 다양한 품종과 지역성       | 토스카나, 피에몬테     |
| 스페인   | 숙성 중심 와인 문화       | 리오하, 리베라 델 두에로 |
| 독일    | 산도 높은 화이트 와인 강점   | 모젤, 라인가우       |
| 포르투갈  | 포트와인 등 주정강화 와인 유명 | 도우루            |
| 오스트리아 | 미네랄감 있는 화이트 와인    | 바하우            |
| 그리스   | 고대 와인 문화 보유       | 산토리니           |

### 구대륙 특징

* 산도와 균형감 중심
* 음식과 함께 마시는 스타일
* 지역 이름 중심 레이블 사용
* 전통 규정 엄격

---

## 신대륙 와인 (New World)

유럽 외 지역에서 발전한 와인 생산국입니다.
과일 풍미와 현대적인 양조 기술이 특징입니다.

| 국가       | 특징              | 대표 지역      |
| -------- | --------------- | ---------- |
| 미국       | 진한 과실향과 현대적 스타일 | 나파 밸리, 소노마 |
| 칠레       | 가성비 우수          | 마이포 밸리     |
| 아르헨티나    | 말벡 품종 유명        | 멘도사        |
| 호주       | 강렬한 쉬라즈 스타일     | 바로사 밸리     |
| 뉴질랜드     | 소비뇽 블랑 세계적 명성   | 말보로        |
| 남아프리카공화국 | 유럽+신대륙 스타일 혼합   | 스텔렌보스      |

### 신대륙 특징

* 과실향 강조
* 알코올 도수 비교적 높음
* 품종 중심 레이블 사용
* 접근성이 좋음

# 2. 와인 종류

## 레드 와인 (Red Wine)

붉은 포도 껍질과 함께 발효한 와인.

### 특징

* 탄닌 존재
* 바디감 강함
* 검붉은 과실 향

### 대표 향

* 체리
* 블랙베리
* 자두
* 초콜릿
* 가죽
* 오크

---

## 화이트 와인 (White Wine)

청포도 또는 껍질 제거 후 만든 와인.

### 특징

* 산도 높음
* 가볍고 신선함
* 차갑게 마심

### 대표 향

* 사과
* 레몬
* 복숭아
* 꽃향
* 허브

---

## 로제 와인 (Rosé Wine)

포도 껍질 접촉 시간을 짧게 하여 만든 분홍빛 와인.

### 특징

* 레드와 화이트 중간 스타일
* 산뜻하고 과실감 풍부

### 대표 향

* 딸기
* 라즈베리
* 수박
* 장미

---

## 스파클링 와인 (Sparkling Wine)

탄산이 포함된 와인.

### 특징

* 청량감
* 축하용 이미지
* 높은 산도

### 대표 종류

* 샴페인
* 프로세코
* 카바

---

## 주정강화 와인 (Fortified Wine)

발효 중 혹은 이후에 증류주를 추가한 와인.

### 특징

* 높은 도수
* 달콤하거나 농축된 풍미
* 숙성향 발달

### 대표 종류

* 포트와인
* 셰리
* 마데이라

---

# 4. 와인 레이블 읽는 법

## 기본 구성 요소

| 항목                | 의미       |
| ----------------- | -------- |
| 생산자(Producer)     | 와이너리 이름  |
| 지역(Appellation)   | 포도 재배 지역 |
| 빈티지(Vintage)      | 포도 수확 연도 |
| 품종(Grape Variety) | 사용 포도 품종 |
| 알코올 도수(ABV)       | 알코올 함량   |
| 용량(Volume)        | 병 용량     |

---

## 구대륙 레이블 특징

### 예시

* Bordeaux
* Bourgogne
* Chianti

### 특징

* 지역명이 크게 표시됨
* 품종 표기가 없는 경우 많음
* 원산지 규정 중요

#### 예시 해석

* Bordeaux → 보르도 지역 와인
* Chablis → 샤르도네 와인 가능성 높음

---

## 신대륙 레이블 특징

### 예시

* Cabernet Sauvignon
* Chardonnay
* Pinot Noir

### 특징

* 품종명이 크게 표시됨
* 이해하기 쉬움
* 브랜드 중심 마케팅

---

## 레이블에서 자주 보는 용어

| 용어             | 의미           |
| -------------- | ------------ |
| Reserve        | 특별 숙성 또는 상위급 |
| Grand Cru      | 최고 등급 포도밭    |
| Estate Bottled | 자가 포도밭 생산    |
| DOC / DOCG     | 이탈리아 원산지 등급  |
| AOC / AOP      | 프랑스 원산지 등급   |
| Brut           | 드라이한 스파클링 와인 |
| Vintage        | 수확 연도 표기     |
