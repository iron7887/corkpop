# 🍷 와인 추천용 품종 데이터 정리

> 설문 추천 시스템에서 사용할 수 있도록
> 각 품종별:
>
> * 향(Aroma)
> * 탄닌(Tannin)
> * 당도(Sweetness)
> * 산미(Acidity)
> * 바디(Body)
> * 추천 음식 (한식 포함)
>   을 기준으로 정리

---

# 1. 🍷 RED WINE

---

## 🍇 카베르네 소비뇽

Cabernet Sauvignon

### 특징

* 향: 블랙커런트, 블랙체리, 초콜릿, 삼나무
* 탄닌: 매우 높음
* 당도: 드라이
* 산미: 중간~높음
* 바디: 풀바디

* 구조감
* 오크향
