# 🍷 Wine Preference Survey & Recommendation Service PRD

## 1. 📌 Overview

### 1.1 Product Summary
사용자의 취향을 기반으로 와인 카테고리와 품종을 추천해주는 설문 기반 웹 서비스

- 1차 설문: 와인 카테고리 + 당도(스위트/드라이) 도출
- 2차 설문: 선택된 카테고리 내에서 품종 추천
- 결과: 개인 맞춤 와인 스타일 + 추천 품종 제공

---

### 1.2 Goal

- 와인 초보자도 쉽게 자신의 취향을 이해하도록 돕는다
- “로제 = 스위트” 같은 오해를 자연스럽게 해소한다
- 간단한 설문으로 신뢰도 있는 추천을 제공한다
