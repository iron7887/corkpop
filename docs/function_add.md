*** 추가기능구현 ***
1. 회원가입/로그인
    1. 로그인 페이지에 회원가입 별도 표시
    2. 회원가입은 이메일, 닉네임, 비밀번호
    3. 회원가입시 기존의 히스토리가 있다면 회원가입한 계정으로 연동 후 임시기록은 삭제.
    4. 비회원의 추천기록은 7일간 저장 후 삭제.
2. 내 와인 기록하기 // 마셔본 와인들을 기록하는 기능, 회원전용.
3. 오늘의 와인 // 오늘의 날씨, 기분, 와인이 필요한 자리 등의 설문을 통해서 와인을 추천.
4. 나의 와인취향을 와인 특징별 펜타그램으로 표시하기. // 산도, 타닌, 당도, 바디,향, 종류
5. 네비게이션 와인이란? 페이지 추가.
    0. 와인 생산국. // 구, 신대륙 별 국가
    1. 와인종류. // 레드, 화이트, 스파클링, 로제, 주정강화 등
    2. 포도품종. // 포도품종 설명
    3. 와인레이블 읽는 법.
