# CorkPop

와인 취향 분석 및 추천 웹 서비스

"CorkPop"은 사용자의 와인 취향을 설문 기반으로 분석하여  
선호할 가능성이 높은 와인 스타일과 품종을 추천하는 프로젝트입니다.

## Features

- 와인 취향 설문 기능
- 품종 및 스타일 기반 추천
- 반응형 웹 UI
- Vercel 배포
- GitHub 연동 및 버전 관리

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel

## Getting Started

```bash
npm install
npm run dev
```

## Project Goal

와인 입문자들도 쉽고 직관적으로  
자신의 취향을 파악할 수 있는 서비스를 목표로 제작하고 있습니다.
